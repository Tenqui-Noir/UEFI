关于 Aether UEFI

Aether UEFI 是一个用于演示启动流程、恢复流程与登录流程衔接关系的固件界面原型。它不对应任何真实出厂设备，而是面向交互展示、视觉复刻与状态流模拟而设计。
项目版本
Aether UEFI Preview 0.9
界面风格
Surface UEFI Inspired
渲染方式
Static HTML / CSS / JavaScript
目标用途
UEFI、WinRE 与 Windows 登录流程演示
当前状态
可交互原型

主要能力

支持设备信息、安全、启动项、日期与时间等页面布局演示
支持启动项勾选、排序与局部闪屏模拟
支持固件设置变化对后续启动行为的扩展设计
适合继续扩展为 WinRE、BitLocker 与登录页联动体验

要设置新的日期和时间，请在“新的日期和时间”编辑框中填写新的日期和时间。填写完成后，按 Enter 提交。
